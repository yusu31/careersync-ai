# CareerSync AI — 要件定義書

作成日: 2026年6月16日  
バージョン: v1.0.0

---

## 1. プロジェクト概要

### 1.1 背景・目的

転職活動において、応募者は以下の構造的な課題を抱えている。

- **情報の分散**: 求人票・口コミ・面接日程・メモが複数サービスに散在
- **比較困難**: 複数企業を同じ軸で比較する手段がない
- **手入力コスト**: 企業情報の整理に時間をかけすぎ、面接対策が疎かになる
- **汎用ツールの限界**: Trello・Notionは転職活動に特化していない
- **直前の焦り**: 面接5分前に「この会社どんな会社だっけ？」が解決できない

### 1.2 コアコンセプト

**「手入力を極限まで排除した究極の参謀」**

企業URLを1つ入力するだけで、AIがWeb上の公開情報を自動収集・分析し、  
スコアリング・比較・面接対策を一元管理できる個人専用サービスを提供する。

## 2. 機能要件

### 2.1 ワンクリック企業AI分析

- ユーザーが企業URL（求人ページ or 公式サイト）を入力する
- バックエンドがバックグラウンドで非同期にWebスクレイピングを実行
- 抽出テキストをGemini 1.5 Proに送信し、以下を自動生成:
  - 企業名
  - 事業概要（3行）
  - 強み（箇条書き）
  - 課題・弱み（箇条書き）
  - 面接想定問答（5問）
  - 評価スコア（5軸 × 10点満点）: 成長性・安定性・カルチャーフィット・WLB・待遇
- AIの解析処理中はローディングアニメーションを表示し、ユーザーの操作を妨げない

### 2.2 スコア比較ダッシュボード

- 登録企業をカード形式で一覧表示
- 各企業に評価スコアのレーダーチャートを表示（Chart.js）
- 全応募状況のファネルチャート表示（検討中→応募済→面接中→内定）

### 2.3 企業横断比較ビュー

- 2〜4社を選択してスプレッドシート風テーブルで項目比較
- ソート・フィルター機能（ステータス・スコア・志望度）
- 各列に企業カラーコードを適用

### 2.4 選考ステータス管理

- ステータス一覧: 検討中 / 応募済 / 書類選考中 / 1次面接 / 2次面接 / 最終面接 / 内定 / 不合格
- ステータスはドロップダウン1クリックで更新
- ステータスに応じて行全体の色が自動変化

### 2.5 面接対策クイックシート

- AIが生成した面接想定問答を企業カードからモーダルで表示
- 各回答テキストにワンクリックコピーボタン
- 「明日面接」バッジ: 翌日に面接がある企業を視覚的に強調表示

### 2.6 スケジュール管理（後期: フェーズ6）

- Googleカレンダーから「面接」「面談」キーワードの予定を自動検知
- ダッシュボードのタイムラインに自動マッピング
- 企業カードへのクイックリンク

### 2.7 ローカルデータ永続化

- 全データをローカルのSQLiteに保存（外部クラウド不使用）
- 転職という個人情報を外部サービスに送信しない

## 3. 非機能要件

| 項目 | 要件 |
|---|---|
| 応答性 | AI解析リクエストは即座に202を返し、バックグラウンドで処理する |
| デザイン | プレミアム・ダークモード（Slate Dark #0f172a ベース）|
| 拡張性 | クリーンなディレクトリ構成でポートフォリオ・GitHubアピールに耐えうる品質 |
| セキュリティ | APIキーは.envで管理し、gitignore対象 |
| 動作環境 | ローカル（Windows/Mac）、ブラウザはChrome推奨 |

### CareerSync AIの差別化ポイント

- URLを貼るだけで150+データポイントをAIが自動収集（他は手入力）
- 日本企業・日本語面接に特化したプロンプト設計
- 全データがローカル保存（セキュリティ重視）
- Googleカレンダーとの双方向同期

## 5. 技術スタック

| レイヤー | 技術 | 選定理由 |
|---|---|---|
| バックエンド | Python 3.11+ / FastAPI | 非同期処理・AIライブラリ親和性・学習コスト低 |
| フロントエンド | HTML5 / Tailwind CSS (CDN) / Vanilla JS | フレームワーク不要・HTMLの延長線上 |
| データ可視化 | Chart.js (CDN) | レーダー・ファネルチャートが容易 |
| データベース | SQLite 3 | Docker不要・ファイル1つでローカル完結 |
| AI | Google GenAI SDK (Gemini 1.5 Pro) | 高品質日本語解析・無料枠あり |
| 外部連携（後期） | Google Calendar API | 面接スケジュール自動検知 |
